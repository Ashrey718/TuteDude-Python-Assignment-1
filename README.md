# TuteDude-Python-Assignment-1

## Description
This repository contains Python programs submitted as part of the TuteDude Python assignment 1.

Programs Included:-

### Task 1: Perform Basic Mathematical Operations (Basic_Math_Operations.py)
- Takes two numbers as input from the user.
- Performs basic arithmetic operations on these two numbers:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Displays the result of each operation on screen.

### Task 2: Create a Personalized Greeting
- Takes a user's first name and last name as input.
- Concatenates the first name and last name into a full name
- Prints a personalized greeting message using the full name.
